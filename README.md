## 📌 Project Overview
This project is a Machine Learning-based Loan Status Prediction system built using Python. The model analyzes applicant data and predicts whether a loan application will be approved or not.

It demonstrates a complete end-to-end classification workflow, including data preprocessing, visualization, model training, and evaluation.

---

## 🎯 Objective
- Predict loan approval status using machine learning
- Understand classification algorithms like SVM
- Gain hands-on experience with real-world financial datasets

---

## 🛠️ Technologies & Libraries Used
- Python
- NumPy
- Pandas
- Seaborn
- Scikit-learn

---

## ⚙️ Machine Learning Workflow
1. **Data Collection**
   - Used a dataset containing applicant financial and personal details

2. **Data Preprocessing**
   - Handling missing values
   - Encoding categorical data
   - Feature selection

3. **Data Visualization**
   - Used **Seaborn** to analyze relationships and patterns in the data

4. **Train-Test Split**
   - Used `train_test_split` to divide the dataset into training and testing sets

5. **Model Building**
   - Implemented **Support Vector Machine (SVM)** for classification

6. **Model Evaluation**
   - Evaluated performance using **accuracy_score**

---

## 📊 Features
- Predicts whether a loan will be approved or not
- Uses SVM for classification
- Includes data visualization for better insights
- Clean and structured ML pipeline
